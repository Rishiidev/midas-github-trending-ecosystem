# Security Policy

## Supported Versions

| Version | Supported |
|---------|-----------|
| 1.x     | ✓ |

## Reporting a Vulnerability

Do not open a public issue for security vulnerabilities.
Open a [GitHub Issue](https://github.com/Rishiidev/midas-github-trending-ecosystem/issues/new)
with the prefix `[SECURITY]` — visible to maintainers only.
